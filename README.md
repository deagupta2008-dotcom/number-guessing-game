# Number Guessing Game

A simple Python game where the user has to guess a randomly generated number.

##  Features
- Random number generation
- User input handling
- Hints (too high / too low)

## ▶️ How to Run
1. Install Python
2. Run the file:
   python game.py

##  Tech Used
- Python (beginner level)
